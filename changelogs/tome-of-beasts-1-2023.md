# Tome of Beasts I 2023 Changelong

## 13.0.0
- This is a complete update to this module from top to tail for Core v13+ and D&D5e 5.2+.
- Removes version gate from the manifest so this module will install on future FoundryVTT Core versions.
- v13+ level Activities and Active Effects (where applicable) added or redone for all creatures.
- All creatures are now organized by creature type in the compendium.
- All Journals creature links are Updated, pointing to the new creature compendium.
- A new compendium of features that were used to create ToB1 creatures are now available to create your own creatures.
- Errors corrected from prior module versions.
- Corrections from the latest Errata document are also included (05/29/2025).
